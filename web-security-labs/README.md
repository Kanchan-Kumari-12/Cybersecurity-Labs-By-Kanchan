# Web Security Labs (Burp Suite)

This repository contains hands-on labs for analyzing web application security using Burp Suite.

##  Labs IncludedR

1. Login Flow Analysis
2. API Traffic Inspection
3. Identifying Security Vulnerabilities

##  Tools Used
- Burp Suite
- Browser DevTools
- Test Web Applications

##  Goal
To understand how web applications handle authentication, API communication, and security risks.
